Perfect Fold Image Stretch plugin

By: Pete Rugh

Introduction

This plugin is a solution to a problem I ran into enough times to necessitate an off-the-shelf solution. Given different browser sizes and in particular, different devices, deciding on where the fold is has become near impossible. This means that when beautiful marquee images are in a design, they need to be cropped to fit most folds. This leads to mixed results and leaves people with either no indication of further content, or an ugly design with a tiny marquee.

Another issue is how to choose how wide to make an image. You can always shoot for 2560px wide (iMac 27"), but file sizes increase as it gets wider, and the audience you are catering to fails to justify the trade off.

These 2 problems are solved by the Perfect Fold Image Stretch plugin. There are many other plugins that let you fill an entire screen vertically and horizontally, but they are not flexible. This plugin achieves what those plugins do, but adds the addition of an offset variable. You can now specify how much content you want showing below a marquee iamge. This means that you can offer the same amount of content to entice users to scroll to regardless of screen size.

So designers are coders can live in harmony knowing that beautiful marquee images now can be enjoyed on all screen sizes while maintaing adherance to the fold rule.

Examples

This plugin, although simple to use, also offers a lot of flexibility. I have created several examples to illustrate how to combine the options to suit your needs.

  • Default Options

    The plugin with no options passed into the object. Image fills entire visible window.

  • Default with vertical offset

    Setting a distance between image and bottom of visible window.

  • Container content

    Using the plugin to set the targeted image as a background for content inside the same container.

  • All options

    Overriding all the default options.

  • Coupled with Cycle Plugin

    This is an example of how you can pair this with another plugin. In this case, the slider plugin jQuery Cycle

  • Vertical Image

    Showing the effect while using an image with a vertical aspect ration.

  • Small Image

    The plugin will stretch a small image to fill the container just as how it shrinks a large image.

Default Values

    • alignHorizontal
    • 'center'
    • Type: String
    • 'left', 'center', 'right'
    • alignVertical
    • 'middle'
    • Type: String
    • 'top', 'middle', 'bottom'
    • changeOnResize
    • true
    • Type: Bool
    • Whether or not to recalculate when resizing viewport
    • container
    • this.parent()
    • Type: Selector
    • Plugin assumes that the parent element is the container.
    • easingMethod
    • 'swing'
    • Type: String
    • jQuery easing method
    • fadeTime
    • 2000
    • Type: Int
    • Image fade in length
    • heightMax
    • null
    • Type: Int
    • Maximum image height
    • heightMin
    • null
    • Type: Int
    • Minimum image height
    • imageLoaded
    • function() {}
    • Type: Function
    • Execute code after all images passed in are loaded. Fires 1 time only.
    • imageVisible
    • function() {}
    • Type: Function
    • Execute code after the images are fully faded in.
    • offset
    • 0
    • Type: Int
    • Amount of space to leave visible below image in viewport. Int is pixels
    • zIndexPlacer

    • 100

    • Type: Int

    • Starting z-index for positiong images on top of each other in the right order. This can help write css for proper z-index layering around the plugin effected image.
